What it is.
Hamsolar is a actually a extremely light weight web browser.
It is dedicated to a single specific and highly specialized web page.
It is not selectable so you cannot follow links.  It displays one of
Paul Herrman's, N0NBH, Solar banners.
http://www.n0nbh.com

I hope you like it.  I use it all the time.
